What is DOM ? What is purpose of DOM?

Meaning of DOM is Document Object Model. To view our element in browser which can be create change and remove. For the Intaractive web pages we use this programming interface.

How to select Element from DOM.

To select element in DOM the main structure is document.methodsname(selector_name)
There are a few methods we can select elements in a DOM__

  1. document.getElementById('Id Name')
  2. document.getElementsByClassName('Class Name')
  3. document.querySelectorAll('Id/class/tag Name')
  4. document.querySelector('Id/class/tag Name')
  5. document.getElementsByTagName('Tag Name')
